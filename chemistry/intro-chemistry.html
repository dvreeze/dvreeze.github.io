Introduction to Chemistry

Introduction

This is a brief introduction to chemistry. It is like a summary, or "cheat sheet". It includes a small introduction to organic chemistry.

This summary is primarily based on the material in Beginning Chemistry.

It is handy to always have a browser tab open with the Periodic Table.

Atoms

Matter is anything that has mass and takes up space. A substance is matter that has the same physical and chemical properties throughout.

An element is a substance that cannot be broken down into simpler chemical substances by ordinary chemical means.

The smallest piece of an element that maintains the identity of the element is called an atom. Hence all matter is made from atoms.

Each atom contains:

  • A compact nucleus containing:
    • Protons. They are positively charged (1+), and weigh about 1.6e-27 kg.
    • Neutrons. They are uncharged, and weigh about the same as protons (slightly more).
  • Electrons. They are negatively charged (1-), and weigh only about 9.1e-31 kg. These electrons circle around the nucleus at a relatively large distance from it.

If the atom is uncharged, it has the same number of protons as electrons.

Periodic Table

All known elements occur in the periodic table, in a purposely chosen layout. For each element, the following data (and more) is found in the periodic table:

  • Name of the element. E.g. Hydrogen, Helium, Carbon, Oxygen, Silver.
  • Element symbol. E.g. (corresponding with the names above) H, He, C, O, Ag.
  • Atomic number. This is the number of protons, and uniquely identifies the element.
  • Key isotopes. While the number of protons is fixed, the number of neutrons can vary for one element, thus leading to different isotopes. E.g. 12C, 13C, 14C.
  • Atomic mass. This is the (average) sum of the number of protons and neutrons. Given that there can be multiple isotopes (some abundantly available, some rare), this is typically not a whole number, but it is a weighted average.
  • Density, in g/cm3.
  • Other physical properties like state at room temperature, melting point, boiling point.
  • Uses and properties, history.

An atomic mass unit (u) is approximately the mass of 1 proton or neutron. More precisely, an atomic mass unit is one-twelfth of the mass of a carbon-12 atom, which is a carbon isotope having 6 protons and 6 neutrons. The mass number of an isotope of an element is the exact number of protons plus the exact number of neutrons (which clearly is a whole number for the isotope). Given that isotopes of an element occur in a given ratio, the atomic mass of an element is the weighted average mass number for that element.

There is more interesting data in the periodic table, but for that we need to know more about electron configurations. In a quantum-mechanical model of an atom, the state of an electron is described by 4 quantum numbers:

  • Principal quantum number, or n. It identifies the shell, which can be thought of as the distance to the nucleus. It is a non-zero whole number: 1, 2 or higher. The higher the number, the further the shell is from the nucleus, and the more energy the electrons in that shell have.
  • Angular momentum quantum number, or l. It identifies the subshell within a shell. It is any number between 0 and n - 1.
  • Magnetic quantum number, or ml. It is any integer between -l and l.
  • Spin quantum number. It is either 1/2 or -1/2.

The Pauli exclusion principle states that no 2 electrons in an atom can have the same set of 4 quantum numbers. Keep that in mind below, when talking about electron configurations. Note that for subshell l (in a given shell) there can be 2 * l + 1 different magnetic quantum numbers, so 2 * (2 * l + 1) electrons (taking the 2 different spin quantum numbers in mind).

The quantum numbers lead to the electron configuration of an element in the periodic table. First give the subshells a letter instead of number, namely s, p, d and f for subshell 0, 1, 2 and 3, respectively. Remember this as SPeeD, Fine, for example. The electron configuration repeats the pattern shell-subshell-(electron-count). For example:

  • 1s1 for H. That is: shell 1, subshell s (or 0), and 1 electron in that subshell.
  • 1s2 for He. That is: shell 1, subshell s (or 0), and 2 electrons in that subshell, which is the maximum number of electrons in any s subshell.
  • [He]2s22p2 for C. Note that [He] stands for 1s2, but there is no need to repeat that (which will become clear later). Also note that for increasing atomic numbers first the lower level shells are filled, roughly, and that per subshell the order is roughly s-p-d-f when "filling electrons".
  • [Xe]4f145d106s1 for Au (gold). All subshells but the last are completely filled.

Note that for uncharged atoms, the number of electrons and protons are the same, which makes sense because protons and electrons have the opposite charge (1 and -1, respectively). Hence, the number of electrons is the same as the atomic number. Note that the number of electrons also easily follows from the electron configuration.

Also note that electrons start to fill the "lowest energy positions" that are not filled. This is clearly visible in the periodic table, for increasing atomic numbers.

The electron configuration of an element tells us a lot about reactivity. When shells and subshells are completely filled, the atom is so stable that it hardly reacts, if at all. That's the case for noble gases, like He, Ne, Ar, Kr and Xe. The octet rule says that atoms like to have 8 electrons in their valence shell. More about that (and exceptions) later.

For each element, the highest numbered shell is called the valence shell. The electrons in the valence shell, plus the electrons in the last unfilled subshell are called valence electrons (but beware of double counting). Elements with similar electron configuration in their valence shells have similar chemistry. This is represented in the periodic table by elements having the same column. For example, the noble gases all have fully filled shells and subshells, and they are all found in the rightmost column.

One trend in the periodic table is that the atomic radius goes up if we go down in one column (group), but goes down if we go to the right in one row (period). The reasons are: going down in one group means an extra shell, increasing the atomic radius, but going to the right in one period means more protons and therefore more positive charge attracting the valence electrons.

The ionization energy is the amount of energy required to remove an electron from an atom in the gas phase. Going down in the periodic table means decreasing ionization energy, because the valence electron is further away from the nucleus, due to it being in a higher numbered shell. Going to the right in the periodic table means increasing ionization energy, due to the increasing pull on the electrons by the increasing number of protons.

There is more than one IE (ionization energy), for removing successive electrons (if the atom has them). These successive IE's increase, due to the larger net positive charge.

The opposite of IE is electron affinity (EA). That is the energy change when a gas-phase atom accepts an electron. The general trend (in kJ/mol) is obvious, but there is no definitive trend.

Electronegativity is a qualitative scale for judging how much an atom of a given element attracts electrons to itself in a chemical bond. Electronegativity (Pauling scale) is the highest for fluorine, that is the top right corner of the periodic table with the exception of the noble gas group (at the last column). It is the lowest at the botton left corner. Obviously, due to the octet rule, halogens (in the column directly left of the noble gases) have high electronegativity. These elements "want to attract" an electron to fully fill their valence shell with 8 instead of 7 electrons. Indeed, going from left to right in the periodic table tends to increase the electronegativity. In left columns, the metals there want to shed electrons rather than attract them, according to the octet rule. When going down, electronegativity decreases, which makes sense in that valence electrons are further away from the positively charged nucleus.

Strictly, electronegativity is a property of an atom within a molecule, rather than an inherent property of an atom.

Types of elements

There are different groups of elements in the periodic table. Most elements are metals. Subgroups are alkali metals, alkaline earths and transition metals, for example. Most metals are solid, shiny, with high density and with high melting points. Mercury (Hg) is liquid at room temperature, however. Most metals have valence electrons that are easily removed, so electronegativity is low. Most metals conduct heat and electricity well.

Nonmetals other than H occur on the upper right side of the periodic table. They have high electronegativity. They poorly conduct heat and electricity. If solid, they are typically brittle. Two subtypes stand out:

  • Noble gases (or: inert gases). They are gases at room temperature. They are the rightmost group in the periodic table. They have a complete valence shell, and are therefore relatively unreactive.
  • Halogens. They have very high electronegativity, and have 7 valence electrons. They are the rightmost group left of the noble gases.

Another group of nonmetals is semimetals (or: metalloids). They are in between metals and nonmetals in the periodic table, and therefore also in electronegativity, ionization energy etc. Reactivity typically depends on the element with which they react.

Molecules, Ions and Ionic Compounds

A molecule is the smallest part of a substance that has the physical and chemical properties of that substance. They combine atoms in whole-number ratios. Molecules have no charge, and their atoms are bound by covalent bonds. That is, their atoms share electrons, instead of one atom stealing the other atom's electrons (which would instead lead to ionic compounds).

There are 2 kinds of molecules:

  • Single-element molecules, or element molecules, or molecular elements.
  • Molecular compounds (being compounds implies that more than 1 element is involved). More precisely, a molecular compound combines 2 or more nonmetals.

Some elements exist naturally as molecules (2-atom or having more than 2 atoms), for example H2, O2, P4 and S8.

Molecules (with a given molecular formula) have a molecular mass which trivially follows from the atomic mass of the elements in the formula and their quantities in the formula.

A compound combines at least 2 different elements. There are 2 types of compounds:

  • Molecular compound. See above. It is a molecule that combines at least 2 different elements.
  • Ionic compound. It is a compound that formed from positive and negative ions. Hence, ionic compounds do not covalently bind atoms (electron sharing), but bind (charged) ions (ionic binding).

Ions are either cations (positively charged, more protons than electrons) or anions (negatively charged, more electrons than protons). Ions form by atoms gaining or losing electrons.

Ignoring the charge (positive or negative), there are 2 types of ions:

  • Monatomic ion. Monatomic cations are hydrogen and metals. Monatomic anions are nonmetals (other than H).
  • Polyatomic ion.

The periodic table helps a lot in understanding ionic compounds. Take for example table salt. It combines Na+ and Cl- ions, through ionic binding. Table salt disassociates in water, breaking it down into its component ions. The periodic table shows that sodium "wants to be" the Na+ ion and chlorium "wants to be" the Cl- ion. After all, given the octet rule and the fact that sodium has one valence electron, it "wants to get rid of" this single electron, giving the resulting ion more of the stability of the neon noble gas. Likewise, chlorium has 7 valence electrons, so it is "more than willing" to receive that extra electron. Similar remarks apply to other element pairs in the same columns in the periodic table, one being a metal and the other one being a nonmetal. So, from the periodic table and the octet rule it is easy to see that sodium ions have charge 1+ (Na+), magnesium ions have charge 2+ (Mg2+), chloride ions have charge 1- (Cl-) and sulfide ions have charge 2- (S2-).

Electronegativity helps in understanding covalent bonds (molecules) vs. ionic bonds. The difference in electronegativity between atoms determines the kind of bond. If this difference is less than 2, the bond is covalent. A difference still makes the molecule polar, where part of the molecule is positively charged, and part of the molecule is negatively charged. If this difference is greater than 2, as is the case between metals and nonmetals, the bond between the atoms is ionic. Recall that in covalent bonds, electrons are shared, whereas in ionic bonds, electrons move from one atom to the other, thus leading to a bond between ions. Organic compounds (having elements such as C, H, O, N etc.) tend to be molecular, due to the electronegativity differences between them being smaller than 2. Note that the difference in electronegativity between F and H is almost 2, so a covalent binding between the 2 elements is highly polar.

Many compounds exist as molecules. Especially, when nonmetals connect with other nonmetals, the resulting compounds are molecules. Binary molecular compounds are named as follows:

  • Use prefixes before the 2 element names (mono-, di-, tri-, tetra-, penta-, hexa-, hepta-, octa-, nona-, deca-) before the element name, but leave out "mono-" before the first element.
  • Change the 2nd element name: chlorine becomes chloride, oxygen becomes oxide, etc.

Take for example the substance with molecular formula SO2 (recall that the elements in the molecular compound occur in whole-number ratios, and 1 is implicit and can therefore be left out in the formula). This substance is called sulfur dioxide. Not surprisingly, SO3 is called sulfur trioxide. Sometimes we use common names instead, like water for H2O, ammonia for NH3 and methane for CH4.

Naming monatomic ions is simple. The cations are simply named as the element (possibly adding the word ion), whereas the anions change names (similarly to binary molecular compounds): chlorine becomes the chloride ion (Cl-) and nitrogen becomes the nitride ion (N3-). If an element has more than one possible charge, this charge becomes part of the name. For example: Fe2+ is the iron(II) cation, and Fe3+ is the iron(III) cation.

Chemical formulas for ionic compounds are called ionic formulas. Like molecular formulas, they show the numbers of atoms of the different elements in the formula. Unlike molecular formulas, ionic formulas have a cation and anion, and the net charge must be 0. For example, take the ionic compound between Mg2+ ions and Cl- ions. To balance the charges, the number of chloride ions must be twice the number of magnesium ions, and the ionic formula is MgCl2. Note that the charges of the ions are implicit and not visible in the ionic formula.

Naming ionic compounds is simple. Combine the names of the ions (leaving out the word "ion"). If the cations can have more than one charge (like iron cations), the charge becomes part of the name of the ionic compound. For example, FeS is iron(II) sulfide (and indeed sulfide has charge 2-), and Fe2S3 is iron(III) sulfide (where again sulfide has charge 2- per ion). So the charge can be part of the name of the ionic compound, whereas it is implicit in the ionic formula.

If molecular formulas and ionic formulas look so much alike, how can we tell them apart? Molecular compounds form between nonmetals and nonmetals, whereas ionic compounds form between metals and nonmetals.

Some well-known polyatomic ions are:

  • Ammonium: NH4+
  • Acetate: C2H3O2-
  • Bicarbonate (hydrogen carbonate): HCO3-
  • Bisulfate (hydrogen sulfate): HSO4-
  • Carbonate: CO32-
  • Chlorate: ClO3-
  • Chromate: CrO42-
  • Cyanide: CN-
  • Dichromate: Cr2O72-
  • Hydroxide: OH-
  • Nitrate: NO3-
  • Nitrite: NO2-
  • Peroxide: O22-
  • Perchlorate: ClO4-
  • Phosphate: PO43-
  • Sulfate: SO42-
  • Sufite: SO32-
  • Triiodide: I3-

Ionic formulas involving polyatomic ions typically need parentheses. For example, Ca(NO3)2 for calcium nitrate, which combines calcium ions (Ca2+) and nitrate ions (NO3-).

Acids are an interesting group of compounds. Initially, they are defined as ionic compounds of the H+ cation dissolved in water. Acids have their own naming scheme. For example, HCl(aq) is called hydrochloric acid, and H2S(aq) is called hydrosulfuric acid. If the anion is polyatomic, we get names like acetic acid (HC2H3O2(aq)), chloric acid (HClO3(aq)), nitric acid (HNO3(aq)) and nitrous acid (HNO2(aq)).

Reactions and Equations

A concise representation of a chemical reaction is its chemical equation. The initial substances are the reactants and the final substances are the products. For example:

2 H2 + O2 ⟶ 2 H2O

This equation is balanced, as it must be. That means that it has the same numbers of atoms (for the different elements involved) left and right of the arrow. In other words, the law of conservation of matter has been satisfied.

A more complex chemical equation would be:

2 NaHCO3(s) ⟶ Na2CO3(s) + CO2(g) + H2O(l)

A single-replacement reaction substitutes one element for another one in a compound. For example:

2 HCl(aq) + Zn(s) ⟶ ZnCl2(aq) + H2(g)

or:

2 NaCl(aq) + F2(g) ⟶ 2 NaF(s) + Cl2(g)

The periodic table helps understand which single-replacement reactions involving halogens will occur and which ones will not occur. Elements higher in the halogen column of the periodic table will replace ones lower in the column, but not the other way around. The elements higher in the halogen column have more electronegativity, so attract the electrons of a bond to a larger extent than elements lower in the halogen column do.

A double-replacement reaction exchanges parts of 2 ionic compounds. For example:

CuCl2(aq) + 2 AgNO3(aq) ⟶ Cu(NO3)2(aq) + 2 AgCl(s)

One type of double-replacement reaction is predictable, namely the precipitation reaction, leaving a solid precipitate. To predict whether some precipitation reaction occurs, the solubility rules are of help. Whatever does not dissolve can be a solid precipitate, and therefore one of the products in a double-replacement reaction.

For the solubility rules, see solubility rules. Also see the following 2 tables:

When molecular compounds dissolve in water, the individual molecules drift apart from each other. Yet when ionic compounds dissolve, the ions physically separate from each other (dissociation). For example:

NaCl(s) ⟶ Na+(aq) + Cl-(aq)

or:

CaCl2(s) ⟶ Ca2+(aq) + 2 Cl-(aq)

Similar remarks apply when polyatomic ions are involved.

When chemicals in a solution react (dissolved ionic compounds), it is therefore most appropriate to write a complete ionic equation, where dissolved ionic compounds are written as separate ions. For example:

Na+(aq) + Cl-(aq) + Ag+ + NO3- ⟶ AgCl(s) + Na+(aq) + NO3-(aq)

The so-called spectator ions (that are the same on both sides of the equation) can be left out, leaving the net ionic equation:

Cl-(aq) + Ag+ ⟶ AgCl(s)

A composition reaction (or combination reaction, or synthesis reaction) produces a single substance from multiple reactants. For example:

2 H2(g) + O2 ⟶ 2 H2O(l)

A decomposition reaction starts from a single substance and produces more than one substance. For example:

2 NaHCO3(s) ⟶ Na2CO3(s) + CO2(g) + H2O(l)

In a combustion reaction a reactant combines with oxygen, producing oxides as products. Any nitrogen in the reactant converts to elemental nitrogen. Many reactants, called fuels, containing mostly carbon and hydrogen atoms, produce CO2 and H2O in the combustion reacion. Examples of combustion reactions:

CH4 + 2 O2 ⟶ CO2 + 2 H2O

2 C12H26 + 37 O2 ⟶ 24 CO2 + 26 H2O

C2H5OH + 3 O2 ⟶ 2 CO2 + 3 H2O

2 C2H2N2O4 + O2 ⟶ 4 CO2 + 2 H2O + 2 N2

An Arrhenius definition of an acid or of a base is as follows: an acid is any compound that increases the amount of hydrogen ion (H+) in an aqueous solution. Example: HCl. A base is any compound that increases the amount of hydroxide ion (OH-) in an aqueous solution. A base is the chemical opposite of an acid. Note that H+ is a proton, but if a hydrogen ion is attached to water molecules, we would represent this as a hydronium ion (H3O+).

A reaction of an acid and a base is called a neutralization reaction. The reactants are the acid and base, and the products are water and a salt. A salt is any ionic compound (whether soluble or insoluble) that is formed from a reaction between an acid and a base. Example neutralization reaction (where the salt is MgCl2):

2 HCl(aq) + Mg(OH)2(aq) ⟶ 2 H2O(l) + MgCl2(aq)

Of course, we could instead have written the complete or net ionic reaction.

Stoichiometry

Stoichiometry is about relating one chemical substance to another using a balanced chemical reaction.

A mole is a number of "things" (without unit) just like a dozen is 12 "things". A mole stands for 6.02214179 * 1023. This number is known as Avogadro's number or Avogadro's constant. This number is equal to the number of atoms in exactly 12 g of carbon-12. In other words, it is about the same as 1 g of protons or neutrons. Put differently, a mole of a substance has the same mass in grams as one unit (atom or molecules) has in atomic mass units. Moles let us express amounts of atoms and molecules in visible amounts that we can understand. In a chemical equation, we can interpret any amount of reactant or product in terms of moles. That is, chemical equations are also balanced in terms of moles. After all, in a balanced equation we can multiply each amount of reactants/products with one and the same factor and the result would still be a balanced equation.

Lewis Electron Dot Diagrams

A Lewis electron dot diagram (or: Lewis diagram) represents valence electrons as dots around the symbol of the element. Per side, there can be no more than 2 dots (which means indeed that there can be no more than 8 valence electrons per atom). Note that Lewis diagrams must correspond to the electron configuration, ignoring all electrons other than the valence electrons.

In Lewis diagrams for monatomic ions we simply add electrons (for anions) or remove electrons (for cations) in the diagram. When representing cations, electrons are first lost from the highest numbered shell, not necessarily the last subshell filled. That's the case in the electron configuration, and in the corresponding Lewis diagram.

Recall the octet rule, stating that atoms like to have 8 electrons in their valence shell. Having 8 electrons in the valence shell is an energetically stable arrangement of electrons.

The octet rule can be violated. Sodium in its elemental form is stable, in spite of it having 1 valence electron. On the other hand, sodium reacts easily, and removing that 1 valence electron only takes little energy, making the Na+ cation. Removing one more electron takes more energy than available to most reactions, however, so in this case the octet rule holds. When a sodium atom meets a chlorine atom, the sodium atom loses its single valence electron to the chlorine atom. Both atoms, that thus became ions, "benefit", in that both ions obey the octet rule. These ions that have opposite charge do attract each other, leading to an ionic bond between them.

How strong is an ionic bond? That depends on the magnitude of the charges and the size of the ion. The greater the magnitude of the charge, the stronger the ionic bond. The smaller the ion, the stronger the ionic bond.

In a covalent bond there is no transfer of electrons, but sharing of electrons instead. Shared electrons also count as valence shell electrons, for both atoms involved in the bond. In a Lewis diagram, it is common to replace the 2 dots of shared electrons by a dash.

Hydrogen is one of the diatomic elements. It makes sense that it is, because hydrogen (when sharing electrons) can have at most 2 electrons in its s subshell. This sharing of 1 pair of electrons is called a single bond. While hydrogen cannot obey the octet rule (it can at best fill the s subshell, by sharing another electron), fluorine can. Fluorine is also diatomic, but when forming the diatomic molecule, both atoms have a complete octet of electrons, even if both shared electrons are counted twice, once per atom.

In a covalent bond, there are bonding electron pairs and lone pair electrons. It is the bonding electron pairs that make the covalent bond. In a Lewis diagram, the bonding electron pairs would typically be drawn as dashes. Each dash is a pair of electrons.

How can we determine a Lewis diagram of a simple molecule? The following steps can be followed:

  1. Determine which is the central atom and which are the surrounding atoms.
  2. Count the total number of valence electrons, increasing the count for negative charges and decreasing the count for positive charges.
  3. Write the central atom and surround it with the surrounding atoms.
  4. Put a pair of electrons between the central atom and each surrounding atom.
  5. Complete the octets around the surrounding atoms (except for hydrogen).
  6. Put remaining electrons, if any, around the central atom.
  7. Check that each atom has a full valence shell.
  8. If that is not the case, try moving electrons by making double bonds etc., and check again for full valence shells.

If a covalent bond is polar, the Lewis diagram can represent that by using "dashes that are much thicker at one end than at the other end". Recall the electronegativity difference scale from nonpolar covalent via polar covalent to ionic. That is, from nonpolar electron sharing via polar electron sharing to electron transfer.

The approximate amount of energy needed to break a covalent bond is called the bond energy of that bond. Breaking a covalent bond is always endothermic, whereas making a bond is always exothermic.

The 3 major types of violations against the octet rule are:

  • Odd-electron molecules, such as NO, NO2 and ClO2.
  • Electron-deficient molecules. Examples: molecules containing beryllium or boron atoms.
  • Expanded valence shell molecules, having more than 8 electrons in the valence shell. For example PF5.

So far molecules were regarded as 2-dimensional constructs. They do have shapes, however, which follows from observed physical properties and chemical behavior. The basis of molecular shapes is VSEPR which stands for valence shell electron pair repulsion. This is not further explained here.

Introduction to Organic Chemistry

Organic chemistry is about studying organic compounds, which are molecules containing carbon. That is, they are compounds containing carbon in covalent binding.

The simplest organic compounds contain only the elements carbon and hydrogen. They are called hydrocarbons. There are 2 main types of hydrocarbons:

  • Aliphatic hydrocarbons. They are hydrocarbons based on chains of carbon atoms.
  • Aromatic hydrocarbons. They have a special 6-carbon ring, called a benzene ring.

Aliphatic hydrocarbons mainly come in 3 types:

  • Alkanes, which have only single covalent bonds.
  • Alkenes, which have at least one C-C double bond.
  • Alkynes, which have at least one C-C triple bond.

The occasional aliphatic hydrocarbons with a ring of C atoms are called cycloalkanes/cycloalkenes/cycloalkynes.

Normal alkanes are the simplest alkanes, having their C atoms bonded in a straight chain. They are named after the number of C atoms, starting with methane, which is a molecule with formula CH4. With increasing numbers of C atoms we get the following names (for normal alkanes):

  • Methane
  • Ethane
  • Propane
  • Butane
  • Pentane
  • Hexane
  • Heptane
  • Octane
  • Nonane
  • Decane

These normal alkanes all obey the formula template CH3(CH2)nCH3 (where n can be 0), except for methane (which is simply CH4).

Alkanes are also called saturated hydrocarbons, because they have the maximum number of H atoms possible according to the rules of covalent bonds.

Alkenes have a C-C double bond, so they are examples of unsaturated hydrocarbons.

Alkenes have names that are similar to alkane names: ethene, propene, butene, pentene etc. Yet how do we express in the name where the double C-C bond is located? That is done by adding a number to the name, e.g. 1-butene and 2-butene. Note that 1-butene and 2-butene have the same molecular formula, namely C4H8, yet they have different physical and chemical properties. Such different molecules having the same molecular formula are called isomers.

Naming for alkynes is quite similar to naming for alkenes, e.g. 1-butyne and 2-butyne.

Aromatic compounds contain the benzene unit. It is a ring of 6 C atoms, with alternating single and double (covalent) C-C bonds. The molecular formula of benzene is C6H6.

Most hydrocarbons are nonpolar because of the close electronegativities of carbon and hydrogen. They dissolve only sparingly in polar solvents such as water. They are also rather unreactive. They do react with halogens. For example:

CH4 + Cl2 ⟶ CH3Cl + HCl

Example of an addition reaction:

(CH2)2 + Cl2 ⟶ (CH2Cl)2

Example of a hydrogenation reaction:

(CH2)2 + H2 ⟶ (CH3)2

Example of a combustion reaction:

2 C8H18 + 25 O2 ⟶ 16 CO2 + 18 H2O

Some hydrocarbons are not straight chains, but are branched. They could still be isomers of straight-chain alkanes, but they do have different chemical and physical properties. They are called branched hydrocarbons. Now do we name them?

Let's first discuss naming for the branches, the substituents. If they are like alkanes, but minus one hydrogen atom, they are called alkyl groups. For example, "methyl-" for methane minus one hydrogen atom, "ethyl-" for ethane minus one hydrogen atom, "propyl-" for propane minus one hydrogen atom, etc. These substituents can substitute individual hydrogen atoms in alkanes, for example.

Consider a molecule which is a hexane, except that it has a branch from the 2nd carbon atom, where the branch replaces the hydrogen atom by a methyl group. Then the name would be 2-methylhexane.

There can be more than 1 branch, though. Take a pentane where the 2nd carbon atom binds to a methyl group and the 3rd carbon atom binds to an ethyl group. This molecule would then be named 3-ethyl-2-methylpentane. Note the alphabetical order of alkyl groups in the name.

Possibly the same alkyl group occurs more than once. Then we need prefixes like "di-", "tri-", "tetra-", "penta-". If there are methyl groups binding to the 2nd and 3rd carbon atom in a butane, we get a 2,3-dimethylbutane. A somewhat more complex branched alkane is 3-ethyl-2,2-dimethylheptane.

For branched alkenes and alkynes a similar naming scheme is used. For example, 2,4-dimethyl-3-heptene.

Naming branched benzene molecules is relatively simple. For example:

  • Chlorobenzene, if one H atom has been replaced by a Cl atom in the benzene ring.
  • Ethylbenzene, if one H atom has been replace by an ethyl group in the benzene ring.
  • 1,3-dichlorobenzene, if 2 H atoms have been replaced by Cl atoms.
  • 1-bromo-2-ethylbenzene, which is like ethylbenzene, but replacing another H atom by a Br atom.

If a benzene ring is considered the substituent in an aliphatic carbohydron, this ring is called phenyl-. For example, 3-phenylpentane.
